# 选择器的基本规则
  * 用于匹配HTML元素
  * 有不同的匹配规则
  * 多个选择器可以叠加
  * 分类
    1. 元素选择器 a{}
    2. 伪元素选择器 ::before{}
    3. 类选择器 .link{}
    4. 属性选择器  [type=radio]
    5. 伪类选择器  ::hover{}
    6. ID选择器 #id{}
    7. 组合选择器 [type=checkbox]+label{}
    8. 否定选择器  :not(.link){}
    9. 通用选择器 *{}
  * 权重
     1. ID选择器 #id{}  +100
     2. 类 属性  伪类  + 10
     3. 元素  伪元素  +1
     4. 其他选择器 +0 
     5. 只要出现优先级高的,优先级就高（不存在数学上的等价关系） 
  * 解析方式和性能
    1. 浏览器从右往左解析,加快css的解析,快速区分哪些不是
